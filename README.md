# Redirección a tu sketch de p5.js

Este repo sirve como enlace “bonito” que redirige a tu sketch del p5.js Web Editor.

## Pasos para publicar en GitHub Pages

1. Sube `index.html` (este archivo ya contiene tu URL de p5.js).
2. Ve a **Settings → Pages** del repositorio.
3. En “Source”, selecciona **Deploy from a branch**.
4. En “Branch”, elige **main** y carpeta **/(root)**. Guarda.
5. Espera 1–2 minutos a que se despliegue.
